# Contributing to No-ATS

This is a non-profit, community-driven project. We maintain high standards for vacancy listings to protect engineers from broken hiring processes.

## How to Post a Vacancy

We don't have a dashboard. We use Pull Requests.

1. Create a new `.md` file in the `/jobs` directory using the following naming convention: 
   `company-role.md` (e.g., `guardidev-senior-rust-engineer.md`).
2. Use the template below. **All fields are mandatory.**
3. GitHub will suggest to make a **Pull Request** — confirm.
4. Submit a **Pull Request**.

### Vacancy Template
```markdown
Position Title e.g. Senior ML Engineer
Company Name e.g. Guardi Dev
Salary Range e.g. $100k - $150k
Location e.g. Remote / City, Country
Posting Date in DD-MM-YYYY e.g. 18-04-2026
Apply Email e.g. hiring@company.com
Tech Stack e.g. Rust, PostgreSQL, etc

===DESCRIPTION===
